# Resume Skill Extractor

- Overview

This project is an AI-powered **Resume Skills Extractor** designed to automatically extract skills from resumes (CVs).  
The system processes PDF resumes, extracts relevant information, and identifies candidate skills using Natural Language Processing (NLP) also NER ( Named Entity Recognition ) 
and Using Bert( Bidirectional Encoder Representations from Transformers ).

The extracted skills can then be used for candidate evaluation and matching with job requirements via Our Application Smart Recruiter ( Récruteur intélligent ).
---

## Features

- Resume parsing from PDF files
- Automatic skill extraction
- NLP-based processing
- Candidate skill scoring
- Integration with backend API
- Designed for recruitment systems
